MPI Commands impi-launchcmd (1MPI)

NAME

impi-launchcmd - remote execution client for IMPI launchd (deprecated)

SYNOPSIS

impi-launchcmd [-host address:port] [-v] [-h] command

DESCRIPTION

This requests an execution of the command argument (i.e., impi-server/impi-relay) to the IMPI launchd whose address and port (address:port) are specified by -host option. See also impi-launchd(1MPI), impi-server(1MPI), and impi-relay(1MPI).

OPTIONS

The following options are supported:

-host address:port
Specifies the contact to the impi-launcher.
-v
Prints verbose messages to the stderr.
-h
Prints simple help.

ENVIRONMENT VARIABLES

EXAMPLES

Example 1

192.10.2.1% export RELAYADDR=192.10.2.1
192.10.2.1% export RELAYPORT=4000
192.10.2.1% impi-launchctl start

192.10.1.1% impi-server -server 2 -port 5000
192.10.1.1:5000

192.10.2.1% impi-launchcmd -host $RELAYADDR impi-relay -relay 0 192.10.1.1:5000 \
             -port $RELAYPORT
192.10.2.1% mpirun -np 4 -client 0 $RELAYADDR:$RELAYPORT ./a.out

192.10.3.1% mpirun -np 4 -client 1 192.10.1.1:5000 ./a.out
