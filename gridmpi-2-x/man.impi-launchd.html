MPI Commands impi-launchd (1MPI)

NAME

impi-launchd - IMPI launcher daemon (deprecated)

SYNOPSIS

impi-launchd [-port port] [-fork] [-log syslog|filename] [-v] [-h]

DESCRIPTION

This accepts and executes a command string (i.e., impi-server/impi-relay) received from IMPI exec. See also impi-exec(1MPI), impi-server(1MPI), and impi-relay(1MPI).

OPTIONS

The following options are supported:

-port port
Specifies the listen port. It is used to establish a connection from an impi-exec.
-fork
Forks myself and let the child process be a background process.
-log filename | syslog
Redirects stderr to the specified file. If syslog is specified, log messages are output to syslog(2).
-v
Prints verbose messages to the stderr.
-h
Prints simple help.

ENVIRONMENT VARIABLES

IMPI_LAUNCHD_PORT=port
Specifies port number to listen.

EXAMPLES

Example 1

192.10.2.1% export RELAYADDR=192.10.2.1
192.10.2.1% export RELAYPORT=4000
192.10.2.1% impi-launchctl start

192.10.1.1% impi-server -server 2 -port 5000
192.10.1.1:5000

192.10.2.1% impi-launchcmd -host $RELAYADDR impi-relay -relay 0 192.10.1.1:5000 \
             -port $RELAYPORT
192.10.2.1% mpirun -np 4 -client 0 $RELAYADDR:$RELAYPORT ./a.out

192.10.3.1% mpirun -np 4 -client 1 192.10.1.1:5000 ./a.out
