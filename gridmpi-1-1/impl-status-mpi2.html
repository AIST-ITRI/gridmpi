NOTES:

  • Note1: GridMPI supports DPM (Dynamic Process Management), but YAMPII does not support DPM yet.
  • Note2: Thread support is only for YAMPII with socket. Others are not tested at all for GridMPI, Vendor MPI, and YAMPII/SCore.
  • Note3: It needs threads. Reconfiguration with threads and using MPI_Init_thread (or setting environment variable _YAMPI_THREADS) is needed.
  • Note4: It only supports NFS mounted file systems (or similar ones with POSIX), currently.
